# VLA Research: Wall-OSS & LWD

This repository contains reproductions and experiments related to Vision-Language-Action (VLA) models.

## Structure

```
├── wall-oss-repro/    # Wall-OSS-0.5 reproduction (4B VLA, gradient-bridged co-training)
├── lwd-repro/         # LWD (Learning While Deploying) reproduction
├── papers/            # Reference papers
│   ├── wall_oss_0.5.pdf
│   └── lwd_paper.pdf
```

## Papers

- **Wall-OSS-0.5** (arXiv:2605.30877): Open-source 4B VLA with zero-shot real-robot capability
- **LWD**: Learning While Deploying - online adaptation for robot policies
